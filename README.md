To run the html file with the bootstrap and jquery, use http://127.0.0.1:3000/html/filename.html
If using Live Server or Live Preview: 
Live Server
    - Open Server: Alt+L, Alt+O (Windows/Linux) or Cmd+L, Cmd+O (Mac).
    - Stop Server: Alt+L, Alt+C (Windows/Linux) or Cmd+L, Cmd+C (Mac).
Live Preview
    - Show Preview Button: Click the "Show Preview" 
    button in the top-right corner of the editor window 
    (it looks like a a magnifying glass or a preview icon).
    - Right-Click Menu: Right-click on an HTML file in the 
    Explorer or editor and select "Show Preview" for an embedded 
    view or "Show Preview in External Browser".
